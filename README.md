# Asquare CRM Frontend

This project is the standalone frontend foundation for the new CRM platform.

## Current scope

- React + Vite + TypeScript baseline
- CRM login shell
- protected routing
- CRM app shell
- API client foundation
- starter dashboard page

## Run locally

1. Copy `.env.example` to `.env`
2. Install dependencies
3. Start the app

```powershell
npm install
npm run dev
```

Default frontend dev URL is `http://localhost:5174` or the next Vite-available port.
